# NIS LZS de/compressor
This tool de/compresses LZS files used in many NIS games (Prinny and many more).

## Install the executable
`npm i -g nis_lzs`  
(Nodejs is required)

## Usage
`nis_lzs decompress <file.lzs> <file.dat>`  
`nis_lzs compress <file.dat> <file.lzs>`
